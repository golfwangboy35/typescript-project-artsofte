# Frontend (`web/`)

Исходный код Angular-приложения **Finance Tracker**. Полное описание проекта, стек, маршруты и правила Git — в [**README в корне репозитория**](../README.md).

## Быстрый старт

```bash
npm install
npm start
```

Откройте `http://localhost:4200/`.

## Скрипты

| Команда | Действие |
|---------|----------|
| `npm start` | Dev-сервер (`ng serve`) |
| `npm run build` | Production-сборка |
| `npm run lint` | ESLint |
| `npm test` | Unit-тесты (Karma) |

Документация Angular CLI: [angular.dev/tools/cli](https://angular.dev/tools/cli).
